# dbarbar.com

My resume page.  Built with Yeoman and by hand.

## Tools used

* Yeoman
* Google Web Fonts
* Zocial icon font
* Hosted on Github using a subtree push

## Building and Deploying

This is here so I don't forget.

````
grunt build
git add/rm ...
git commit ...
git subtree push --prefix dist origin master
````
